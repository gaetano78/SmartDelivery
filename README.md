# SmartDelivery
Repository for the MSCA project SmartDelivery.

In the CVRP-solver folder you can find a python script to solve a generic, travel time-based CVRP instance. You can find also a test CVRP instance to see how the script works.  

At the following link you can find a simple streamlit-based dashboard which is based on the script you find in the CVRP-solver folder. The dashboard is uploaded in the Huggingface space, so the available computational power is limited.

https://gaetanoldg-smartdelivery-cvrpsolver.hf.space/

In the CVRP-generator folder you can find a a python script to generate generic, travel time-based CVRP instances. You can change several parameters in order to generate different CVRP instances. Generated instances can be solved using the python script you find in the CVRP-solver folder. At the following link you can find a simple streamlit-based dashboard which is based on the script you find in the CVRP-generator folder. The dashboard is uploaded in the Huggingface space, so the available computational power is limited.

https://gaetanoldg-cvrp-generator.hf.space/

You can also find also a 10.000 CVRP instances dataset generated with the scripts you find in the repository. The entire dataset is available in the releases, here in the repository, or you can easily download it by cliking the following link:

https://github.com/gaetano78/SmartDelivery/releases/download/T-CVRP_v1.0_dataset_directories/T-CVRP.v1.0_dataset_directories_structure.zip

If you unzip the file, you will have the entire dataset organized in a clean structure with directories and subdirectories. You will also find, for each instance, a file containing useful information and statistics about each instance.

In the releases you will also find all the instances (.vrp), with no directories' structure, just a list of files containing the instances. If you prefer, you can easily download it by cliking the following link:

https://github.com/gaetano78/SmartDelivery/releases/download/T-CVRP_V1.0_dataset_no_directories_no_statistics/dataset_no_directories_no_statistics.zip

You can also find the dataset on ZENODO: https://zenodo.org/records/18415031
