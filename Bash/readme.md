- Here you can find the script to generate a bash file that let you create a balanced set of 10.000 CVRP instances whose cost matrix is a travel time matrix.

- Of course you need to specify the name of the CVRP instance generator file, which in the script is considered as "LDG_Cleaned_Generatore_istanze_CON_TRAVEL_TIMES.py".

- You can also find an example of generated .sh file.

- By executing the bash file, you will call the generator script 10.000 times, and you will generate 10.000 instances.
